Text Fields Draft

Use text fields to help people enter, select and search for text.

Default single line

<input class="tuiTextField" type="text" />
<input class="tuiTextField" type="text" value="With some text..." />

Place holder

Informs users what they need to do to interact with the field (e.g. 'Start typing to see names'), or shows an example of a typical entry (for example, 'e.g. My first project'). Examples must start with 'e.g.' Only supply placeholder text where clarification is required, do not overuse it.

<input class="tuiTextField" type="text" placeholder="Placeholder text" />

Disabled

Only show disabled fields if users need to know that the control might be available to them in other scenarios.

<input class="tuiTextField" type="text" disabled="disabled" />
<input class="tuiTextField" type="text" placeholder="Placeholder text" disabled="disabled" />
<input class="tuiTextField" type="text" value="With some text..." disabled="disabled" />

Default multi line

<textarea class="tuiTextField"></textarea>
<textarea class="tuiTextField">With some text...</textarea>

Place holder

Lets people know how they should use the field (e.g. 'Start typing to see names') or you can show an example (e.g. 'My first project'). Only supply where clarification is required, try not to overuse it.

<textarea class="tuiTextField" placeholder="Placeholder text"></textarea>

Disabled

Only show disabled fields if users need to know that the control might be available to them in other scenarios.

<textarea class="tuiTextField" disabled="disabled"></textarea>
<textarea class="tuiTextField" placeholder="Placeholder text" disabled="disabled"></textarea>
<textarea class="tuiTextField" disabled="disabled">With some text...</textarea>

Resize

Multi line textboxes must remain resizable at all times. It can be restricted to vertical only resize but there should always be a way to make the textbox larger.

<textarea class="tuiTextField">Default resize</textarea>
<textarea class="tuiTextField tuiTextField--vertical-only">Vertical only resize</textarea>

Guidelines

Length

The length of the field should communicate the intended length of content. These lengths also apply for other form controls of a similar shape, like drop-down menus and text boxes.

Accessibility

Auto focus the first field by default. Users should then be able to tab through elements in the form in a logical sequence. This maintains Section 508 accessibility compliance.
